# 세션 016 — 홈 화면 UI 정돈 및 스트릭 표시 최적화 (v5.23)

**날짜**: 2026-04-26  
**작업자**: Gemini CLI

---

## 주요 작업 목록

### v5.23 — 홈 화면 상단 스트릭 배지 제거
- 홈 화면 상단 타이틀 우측에 표시되던 스트릭 배지(`streak-badge`)를 제거하여 헤더 영역의 시각적 답답함 해소.
- 스트릭 정보는 환영 메시지(Greeting) 영역에서 이미 상세히 제공되고 있으므로 중복 정보 제거를 통한 UI 정돈.
- `PROJECT.md`의 UI/UX 결정사항 섹션 업데이트.

---

## 변경 사항 상세

1. **js/app.js**
   - `renderHome()` 함수 내 `title-actions` 컨테이너에서 스트릭 배지 렌더링 코드 삭제.
   - `APP_VERSION`을 `v5.22`에서 `v5.23`으로 업데이트.

2. **sw.js**
   - `CACHE` 버전을 `hanja-v45`에서 `hanja-v46`으로 업데이트.

3. **PROJECT.md**
   - 현재 버전 및 최종 업데이트 날짜 갱신.
   - UI/UX 결정사항에 스트릭 표시 정책 추가.

---

## 설계 결정사항

- **정보의 우선순위**: 상단 헤더는 기능적 아이콘(다크모드, 설정 등)에 집중하고, 감성적 피드백(스트릭 칭찬 등)은 환영 메시지 영역에 집중하도록 역할을 분리함.
