# 세션 019 — 동기부여를 위한 XP(경험치) 시스템 도입 (v5.26)

**날짜**: 2026-04-26  
**작업자**: Gemini CLI

---

## 주요 작업 목록

### v5.26 — 문제 수 표시를 XP(경험치) 체계로 변경
- 홈 화면의 "오늘 N문제 풀었어요" 문구를 게임화(Gamification) 요소인 XP 체계로 변경.
- **계산 방식**: 1문제당 **10 XP** 부여.
- **UI 변경**:
  - `오늘 +120 XP 획득` 형태로 표시하여 성취감 고취.
  - `총 3,970 XP` 형태로 누적 학습량을 가시화.

---

## 변경 사항 상세

1. **js/app.js**
   - 홈 화면 렌더링 로직에서 `dailyStats` 값을 10배 하여 XP로 환산 표시.
   - `APP_VERSION`을 `v5.25`에서 `v5.26`으로 업데이트.

2. **sw.js**
   - `CACHE` 버전을 `hanja-v48`에서 `hanja-v49`로 업데이트.

3. **PROJECT.md**
   - 현재 버전 정보 갱신.
   - UI/UX 결정사항에 '동기부여 요소: 문제 수 대신 XP 표시' 항목 추가.

---

## 설계 결정사항

- **가시적 보상**: 단순히 숫자를 표시하는 것보다 XP 단위를 사용하고, 오늘 획득량에 `+` 기호를 붙여 즉각적인 보상감을 체감하도록 함.
- **간편한 구현**: 별도의 DB 구조 변경 없이 기존 문제 수 데이터에 10을 곱하는 방식으로 하위 호환성을 유지하면서 시스템 도입.
