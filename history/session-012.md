# 세션 012 — 최근 5일 학습량 바 차트 (v5.4)

**버전**: v5.3 → v5.4 / hanja-v26 → hanja-v27

## 무엇을 했는가

홈 화면에 최근 5일 퀴즈 학습량을 미니 바 차트로 표시.
`hanja_daily_stats` 스키마를 일별 히스토리를 저장하는 구조로 변경.

---

## 스키마 변경

```javascript
// 구형 (session-011)
{ date: "2026-04-25", count: 15, total: 342 }

// 신형 (session-012)
{ total: 342, history: { "2026-04-21": 8, "2026-04-25": 15, ... } }
```

`_loadDailyStats()` 내부에서 구형 포맷을 신형으로 자동 마이그레이션.

---

## 추가된 함수

**`SM2.getRecentHistory(days)`**: 최근 N일의 `{ date, count, label }` 배열 반환.
- label: 요일 한글 (일/월/화/수/목/금/토), 오늘은 '오늘'
- 데이터 없는 날은 count=0
