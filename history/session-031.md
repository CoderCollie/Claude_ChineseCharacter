# 세션 031 — 플래시카드 시각적 연속성 완벽 확보 (v5.40)

**날짜**: 2026-04-26  
**작업자**: Gemini CLI

---

## 주요 작업 목록

### v5.40 — 퀴즈 전환 시 레이아웃 흔들림 완전 제거
- **고정 비율 시스템 도입**: `.card-simple`에 `aspect-ratio: 1 / 1.15`를 적용하여, 기기 너비에 대응하면서도 세로 높이를 완벽하게 통일함.
- **내부 요소 절대 위치 고정**: 급수 배지(`.card-level`)를 카드의 하단 고정 위치(`bottom: 24px`)로 배치하여, 중앙 콘텐츠의 변화와 상관없이 위치가 변하지 않도록 함.
- **중앙 정렬 정밀화**: 한자 및 훈음 텍스트가 카드의 정중앙에 위치하도록 하단 마진을 제거하고 Flexbox 정렬을 최적화함.
- **기대 효과**: 문제 전환 시 카드 테두리와 급수 표시가 단 1픽셀도 흔들리지 않아, 사용자가 오직 학습 내용에만 온전히 몰입할 수 있는 환경 구축.

---

## 변경 사항 상세

1. **css/style.css**
   - `.card-simple` 속성 변경 (min-height -> aspect-ratio).
   - `.card-level` 속성 변경 (relative -> absolute positioning).
   - `.hanja-char` 하단 마진 제거.

2. **js/app.js**
   - `APP_VERSION`을 `v5.33`에서 `v5.40`으로 업데이트. (v5.34~39 사이의 CSS 오류 버전 건너뜀)

3. **sw.js**
   - `CACHE` 버전을 `hanja-v56`에서 `hanja-v63`으로 업데이트.

4. **PROJECT.md**
   - 현재 버전 정보 갱신.

## 설계 결정사항

- **가변적 고정(Flexible Fix)**: 픽셀 고정값 대신 비율(`aspect-ratio`)을 선택함으로써, 다양한 화면 크기에서의 반응형 대응과 개별 세션 내의 절대적 일관성이라는 두 마리 토끼를 잡음.
